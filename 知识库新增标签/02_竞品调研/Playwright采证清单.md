# Playwright 采证清单：知识库新增标签

状态：待登录火山云/阿里云控制台后执行

## 采证前准备

- 不在聊天或文件中记录账号密码、Cookie、Token、API Key。
- 用户在内置浏览器或独立 headed 浏览器中完成登录。
- 如需 Network/HAR，只保存脱敏后的 URL、方法、路径、schema 和错误结构。
- 不点击会创建真实数据、删除真实数据、发送真实请求的按钮，除非用户明确授权环境和测试数据。
- 每个截图必须同时记录：竞品、产品线、账号角色、区域/环境、URL、页面标题、采集时间、视口、前置动作。

## 登录后立即执行顺序

### A. 火山云 / 火山引擎

1. 进入火山知识库或火山方舟知识库控制台，记录产品入口、区域、账号角色。
2. 找到“创建知识库 / 高级配置 / 标签配置”相关入口，只读查看标签字段类型、版本限制、创建后是否可改。
3. 进入一个已有或测试知识库的“原始文档 / 文档列表 / 文档详情”，查看是否有标签列、编辑入口、批量操作入口。
4. 在不提交真实变更的前提下打开编辑弹窗，记录字段、校验、保存/取消、删除、空态、无权限态。
5. 进入知识检索、问答测试或服务调用配置，查看 `doc_filter` 是否有控制台配置入口。
6. 进入费用、资源组或项目相关页面，记录知识库资源和费用归属是否可与项目/成本中心关联。
7. 若某一步页面行为依赖接口，再单独打开 Network，清空后执行一个只读或用户授权的动作，保存脱敏结构。

### B. 阿里云 / 百炼 / PAI

1. 进入百炼或 PAI 知识库控制台，记录产品入口、区域、账号角色。
2. 进入数据导入/数据管理页，查看上传时设置标签、已上传文档编辑标签的入口。
3. 进入知识检索服务配置，查看多知识库绑定、知识库路由、知识库独立参数和“标签过滤”。
4. 进入智能体应用或 API 调用配置，查看 `tags` 参数或标签过滤在应用侧的配置位置。
5. 进入计费、配额、存储或调用量页面，记录标签过滤是否和费用相关；若仅知识库资源计费，也记录为 `INFERRED/TO_CONFIRM`。
6. 需要接口证据时，只采标签过滤相关请求结构，不保存 Authorization、Cookie、租户 ID、文档正文。

## 每个 Flow 的最低证据包

| 内容 | 要求 |
|---|---|
| Screenshot | 至少 1 张页面全局截图；弹窗/错误/空态需单独截图 |
| UI notes | 入口、展示字段、默认值、按钮、保存/取消、刷新后状态 |
| Network | 仅当页面字段无法从 UI 判断，或 PRD/API 需要字段名时采；必须脱敏 |
| Evidence row | 写入 `evidence/evidence-index.md`，状态只能是 `VERIFIED/INFERRED/TO_CONFIRM` |

## 停止与确认规则

- 登录、验证码、MFA、权限申请由用户处理；不在聊天中发送密码或验证码。
- 看到会创建、删除、保存、发送请求、开通计费、购买资源的按钮时，先停下，除非用户已明确授权该具体动作和测试数据。
- 页面里如果出现“上传文档”要求，优先使用用户提供的测试文件；没有测试文件时只记录入口，不上传真实业务文件。
- 若火山/阿里控制台实际页面与公开文档矛盾，以授权控制台截图和时间为准，并在证据台账标记冲突。
