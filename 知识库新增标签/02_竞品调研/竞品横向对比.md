# 竞品横向对比：知识库标签与文档标签

状态：已完成 Playwright 补采（2026-07-13）

正式竞品：火山方舟、阿里云百炼/PAI。

## 1. 总览

知识库级标签和文档级标签是两个独立对象。火山方舟同时提供知识库级“费用标签”和强类型文档标签；阿里云百炼当前控制台与接口只提供上传数据/文档标签，没有知识库级标签。

知识库级标签用于整个知识库的资源分类和项目分账；文档级标签用于知识库内文档分类，并在权限范围内进一步缩小召回文档和切片。

| 对比对象 | 知识库级标签 | 文档级标签 |
| --- | --- | --- |
| 作用范围 | 整个知识库 | 知识库内的单篇或多篇文档 |
| 主要用途 | 资源分类、项目分账、知识库筛选 | 文档分类、检索前过滤 |
| 计费边界 | 火山明确用于项目分账 | 不参与计费 |
| 金山云入口 | 新建知识库、知识库列表或基本信息 | 导入文档、文档信息、知识库设置、知识库检索 |
| 本期数据模型 | 字符串标签，与知识库绑定 | 字符串标签，与文档绑定；强类型字段归元数据需求 |

### 2.2 后端接口对比

| 能力 | 火山方舟 | 阿里云百炼/PAI | 金山云最小改造 |
| --- | --- | --- | --- |
| 知识库详情 | `GetCollectionInfo` 响应包含知识库级 `tags[]`；空数组对应页面“费用标签：-” | `rag-knowledgebase.detail` 响应包含索引、Meta、数据源、检索与规格配置，不包含知识库级 `tags` | 知识库详情增加可选标签数组；不传或为空保持现状 |
| 知识库列表 | `ListCollections` 加载列表；当前列表 UI 不展示标签摘要 | `rag-knowledgebase.list` 响应不包含知识库级 `tags` | 列表响应增加标签摘要，并支持可选标签筛选 |
| 标签键查询 | `GetTagKeys` 查询自定义标签键，参数含 `TagType=custom`、匹配方式、分页 | 知识库模块没有对应查询；标签查询用于上传数据 | 复用资源标签键查询或由星流适配，支持关键字与分页 |
| 标签值查询 | `GetTagValues` 按多个标签键查询值和绑定资源数，参数含 `CountResource=true` | 文档标签通过 `dataHub.getTagsHistoryByTenant` 查询历史候选 | 知识库标签与文档标签使用不同候选源，不共用绑定接口 |
| 定义与绑定 | 资源管理页面明确支持通过云产品控制台或 OpenAPI 创建标签并绑定资源，也提供解绑和删除 | 知识库模块没有知识库级标签定义与绑定接口 | 优先把星流知识库接入金山云资源标签；保留绑定、解绑和操作审计 |

## 3. 文档级标签

### 3.1 前端控制台对比

| 流程/页面 | 火山方舟旗舰版 | 阿里云百炼/PAI | 金山云取舍 | 截图证据 |
| --- | --- | --- | --- | --- |
| 标签定义 | 创建知识库高级配置中定义“文档标签”，最多 180 个；包含名称、类型和可选项 | 创建知识库第 2 步上传数据时新增或复用字符串标签，最多 100 个 | 不在金山云“新建知识库”页定义文档标签；在知识库设置维护可复用字符串标签 | 截图文件：evidence/screenshots/TAG-EV-0025-volc-ark-create-kb-doc-tags-section.png；evidence/screenshots/aliyun/ALIYUN-TAG-0013-create-kb-step2-document-tags.png |
| 数据模型 | 强类型 Schema，支持 List、数值、Bool、DateTime、GeoPoint | 字符串标签，每个最多 32 个字符 | 文档标签只使用字符串；强类型能力归元数据需求 | 截图文件：evidence/screenshots/TAG-EV-0027B-volc-ark-create-kb-doc-tag-field-types-viewport.png；evidence/screenshots/aliyun/ALIYUN-TAG-0002-create-kb-upload-tag-input.png |
| 导入时赋值 | 旗舰版文档基于创建时定义的标签 Schema 赋值；标准版本地上传表单不提供标签字段 | 上传文件区域直接设置本批文档标签 | 在现有“导入文档”页上传文件后设置标签，应用到本批文档 | 截图文件：evidence/screenshots/TAG-EV-0023-volc-ark-local-upload-form.png；evidence/screenshots/aliyun/ALIYUN-TAG-0013-create-kb-step2-document-tags.png |
| 导入后编辑 | 原始文档列表“标签”列逐篇编辑 | 文档列表“更多 > 标签”后置编辑 | 在“文档信息”支持单篇编辑和批量添加或移除 | 截图文件：evidence/screenshots/TAG-EV-0021-volc-ark-doc-tag-edit-dialog.png；evidence/screenshots/aliyun/ALIYUN-TAG-0005-doc-tag-management-dialog.png |

### 3.2 后端接口对比

| 能力 | 火山方舟 | 阿里云百炼/PAI | 金山云最小改造 |
| --- | --- | --- | --- |
| 标签 Schema | `CreateCollection` 创建旗舰版知识库时提交强类型文档标签 Schema | 上传阶段使用租户历史标签候选，不建立强类型 Schema | 文档标签保持知识库内去重的字符串集合；强类型字段由元数据接口承担 |
| 导入并打标 | `AddDoc` 导入文档；标签值受知识库 Schema 约束 | `dataHub.createDataListReturnFileIds` 的数据项与 `tags[]` 同传 | `ImportDocuments` 增加可选 `tags[]`；不传时行为不变 |
| 文档列表 | `GetKnowledgeDocList` 返回文档记录和 `Meta` 字段 | `rag-knowledgebase.dataList` 的文档项明确包含 `tags` | `DescribeDocuments` 可选返回 `tags[]`，保留分页、状态和重索引结构 |
| 单篇更新 | 官方更新文档信息能力和控制台标签编辑入口均已验证 | `dataHub.updateTag` 传文档标识与完整 `tags[]` | 新增单文档标签替换或增删接口，支持幂等和版本冲突校验 |
| 候选标签 | 来自旗舰版知识库预定义 Schema | `dataHub.getTagsHistoryByTenant` 查询租户历史文档标签 | 知识库内标签去重聚合，支持关键字搜索和分页 |
| 保存过滤 | `CreateSearchConfig` 提交 `Filter.Relation + Conditions[]` | `rag/app/update` 写入 `search_filters[].tags` | `RetrieveKnowledge` 增加可选标签条件组；不传时保持原召回结果 |
| 执行检索 | `GetKnowledgeServiceChat` 使用 `FilterMode=tag` | `indices/knowledge/search` 执行标签过滤检索 | 权限过滤先执行，再按标签过滤候选文档和切片 |

## 4. 实测请求结构

火山文档标签过滤结构：

```json
{"Filter":{"Relation":0,"Conditions":[{"FieldName":"liststring","FieldType":"list<string>","Operator":0,"Values":["product"]}]},"FilterMode":"tag"}
```

阿里文档标签过滤结构：

```json
{"kb_search_configs":[{"search_filters":[{"tags":["product"]}]}]}
```
