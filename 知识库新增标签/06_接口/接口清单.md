## 已验证

当前浏览的瀚海 OPEN-API 元数据目录中未发现标签接口；金山云 API Explorer / 星流平台当前知识库接口中也未发现标签 CRUD、绑定/解绑或标签检索过滤字段。不得由元数据接口名称、通用 REST 习惯或竞品实现推造我方标签路径。

## 金山云当前平台 API Explorer 核查

入口：`https://apiexplorer.ksyun.com/#/document/documentList/239/1246`

当前星流平台「模型应用 > 知识库」已公开的知识库接口包括：

| 能力 | Open API | 与标签需求的关系 |
|---|---|---|
| 查询模型列表 | `DescribeKnowledgeBaseModels` | 无知识库级/文档级标签字段 |
| 开通知识库服务 | `ActivateKnowledgeBaseService` | 无标签字段 |
| 知识库检索 | `RetrieveKnowledge` | 当前参数含 `DatasetId`、`Query`、`RetrievalModel`、`TopK`、`ScoreThreshold` 等；未见标签过滤字段 |
| 获取切片详情 | `DescribeChunk` | 无标签字段 |
| 文档索引/嵌入状态 | `BatchDisplayStatus`、`DisplayStatus`、`IndexingStatus` | 无标签字段 |
| 文档管理 | `ImportDocuments`、`DescribeDocuments`、`DescribeDocument`、`DeleteDocument`、`ReindexDocuments`、`ModifyDocumentStatus` | 当前未见文档级打标、批量打标、标签列表或标签过滤字段 |

结论：截至 2026-07-10 只读核查，当前平台开放接口无法证明标签能力已上线。知识库级标签还涉及业务平台/计费归属，必须补齐业务平台接口或同步契约后才能进入正式原型；文档级标签也需要知识库侧标签接口后才能作为可落地功能写入 PRD。

| 竞品 | 能力 | 已知字段/参数 | 证据 |
|---|---|---|---|
| 火山云/火山引擎 | 文档标签过滤 | `doc_filter` 用于检索阶段的标签权限过滤 | `https://docs.volcengine.com/docs/84313/2192050?lang=zh` |
| 火山云/火山引擎 | 文档打标 | 导入文档时传入标签字段，或上传后通过更新文档信息接口设置标签字段 | `https://docs.volcengine.com/docs/84313/2192050?lang=zh` |

## 参考实现：Dify Knowledge Tags API

> 仅作为非竞品参考实现，不等同于正式竞品证据，也不等同于我方接口设计。

| 能力 | 方法/路径 | 已见字段 | 证据 |
|---|---|---|---|
| 创建知识库标签 | `POST /datasets/tags` | `name`；响应 `id/name/type/binding_count` | `https://docs.dify.ai/en/api-reference/tags/create-knowledge-tag` |
| 查询全部知识库标签 | `GET /datasets/tags` | 响应 `id/name/type/binding_count` | `https://docs.dify.ai/en/api-reference/tags/list-knowledge-tags` |
| 更新知识库标签 | `PATCH /datasets/tags` | `tag_id/name` | `https://docs.dify.ai/en/api-reference/tags/update-knowledge-tag` |
| 绑定知识库标签 | `POST /datasets/tags/binding` | `tag_ids[]/target_id`，204 成功 | `https://docs.dify.ai/en/api-reference/tags/create-tag-binding` |
| 解绑知识库标签 | `POST /datasets/tags/unbinding` | `target_id/tag_ids[]`，`tag_id` 为旧单标签字段 | `https://docs.dify.ai/en/api-reference/tags/delete-tag-binding` |
| 查询某知识库标签 | `GET /datasets/{dataset_id}/tags` | 响应 `data[]/total` | `https://docs.dify.ai/en/api-reference/tags/get-knowledge-base-tags` |
