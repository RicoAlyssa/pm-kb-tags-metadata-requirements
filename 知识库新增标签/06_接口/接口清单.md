# 接口清单：知识库新增标签

状态：评审稿。完整字段、请求/响应和错误语义见 [后端接口契约-最小改动.md](后端接口契约-最小改动.md)。

## 1. 当前平台基线

来源：金山云 API Explorer，只读抓取，2026-07-14。

当前知识库相关 Action：

- `ImportDocuments`
- `DescribeDocuments`
- `DescribeDocument`
- `RetrieveKnowledge`
- `DeleteDocument`
- `ReindexDocuments`
- `ModifyDocumentStatus`

结论：

- 当前平台未暴露标签定义、标签绑定、标签过滤字段。
- 文档级标签需要新增标签池和绑定接口。
- 知识库级标签涉及业务/计费归属，需要业务平台协作；文档级标签不走计费链路。
- 检索过滤建议在现有 `RetrieveKnowledge.RetrievalModel` 下兼容扩展，不新建检索 API。

## 2. 最小新增/适配 Action

统一入口：

```text
POST http://aicp.api.ksyun.com/?Action=<ActionName>&Version=2025-11-14
```

## 3. P0 字段和语义

| 能力 | P0 结论 |
|---|---|
| 知识库级标签 | 有 `Name`、`BusinessOwner`、`BoundDatasetCount`；可返回同步异常，但成功态不常驻展示 |
| 文档级标签 | 有 `Name`、`BoundDocumentCount`、`FilterReferenceCount`；不含计费字段 |
| 批量打标 | 部分成功，返回成功项和失败项 |
| 删除文档标签 | 被检索配置引用时阻断；仅绑定文档时允许删除并移除绑定 |
| 删除知识库级标签 | 被知识库/应用/计费对象引用时阻断 |
| 导入打标 | `DocumentTagIds` 可选；标签失败建议整体失败，避免半成功 |
| 检索过滤 | `TagFilter.IncludeTagIds`、`ExcludeTagIds`、`IncludeMatchMode` |
| 权限 | 文档权限先收敛候选范围，标签只能继续缩小，不能扩大权限 |

## 4. 前端可实现性

| 前端区域 | 后端支撑 |
|---|---|
| 基本信息知识库标签 | `DescribeKnowledgeBaseTags(DatasetId)` + `ModifyKnowledgeBaseTagBindings` |
| 文档标签池 | `DescribeDocumentTags` |
| 新建/编辑/删除文档标签 | `CreateDocumentTag` / `ModifyDocumentTag` / `DeleteDocumentTag` |
| 导入时添加文档标签 | `ImportDocuments.DocumentTagIds` |
| 文档列表标签列 | `DescribeDocuments.IncludeDocumentTags=true` |
| 单文档编辑标签 | `DescribeDocument.IncludeDocumentTags=true` + `BatchModifyDocumentTags` |
| 批量设置标签 | `BatchModifyDocumentTags` |
| 召回测试标签过滤 | `RetrieveKnowledge.RetrievalModel.TagFilter` |

## 5. 保留给后端/业务平台评审的命名项

以下不是需求空缺，而是实现侧最终拍板：

- 知识库级标签是否复用公司级资源标签/费用标签主数据源。
- `BusinessOwner.Type` 的最终枚举。
- 同步失败是否由星流直接重试，还是跳转业务平台处理。
- 批量最大数量和标签数量上限。
