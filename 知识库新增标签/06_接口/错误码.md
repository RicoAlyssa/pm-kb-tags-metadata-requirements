# 错误码：标签

状态：候选错误模型；真实 code 待接口确认

| 场景 | HTTP/业务码 | 触发条件 | 前端提示 | 恢复方式 | 状态 |
|---|---|---|---|---|---|
| 未登录/密钥无效 | 401 / TO_CONFIRM | 登录态或 API Key 无效 | “登录已失效，请重新登录” | 重新登录 | TO_CONFIRM |
| 无权限 | 403 / TO_CONFIRM | 无标签管理、知识库管理或业务平台权限 | “你没有操作该标签的权限” | 申请权限/切换账号 | TO_CONFIRM |
| 标签不存在 | 404 / TO_CONFIRM | 编辑/删除/绑定已删除标签 | “标签不存在或已被删除，请刷新” | 刷新列表 | TO_CONFIRM |
| 知识库/文档不存在 | 404 / TO_CONFIRM | 绑定目标不存在 | “目标不存在或已被删除” | 刷新目标列表 | TO_CONFIRM |
| 标签重名 | 400 / TO_CONFIRM | 同一作用域下名称重复 | “标签名称已存在” | 修改名称 | TO_CONFIRM |
| 标签名称非法 | 400 / TO_CONFIRM | 超长、空值、非法字符 | “请输入符合规则的标签名称” | 修改输入 | TO_CONFIRM |
| 绑定超过上限 | 400 / TO_CONFIRM | 单对象绑定标签数超限 | “标签数量已达上限” | 移除部分标签 | TO_CONFIRM |
| 标签被引用无法删除 | 409 / TO_CONFIRM | 标签仍绑定知识库/文档/应用/计费对象 | “该标签仍被引用，解除后再删除” | 查看引用并解除 | TO_CONFIRM |
| 业务平台同步失败 | 502/503 / TO_CONFIRM | 知识库级标签同步计费归属失败 | “标签已保存，但计费同步失败，可重试” | 重试/联系管理员 | TO_CONFIRM |
| 批量部分失败 | 207/200+failed_items / TO_CONFIRM | 部分文档或知识库绑定失败 | “部分处理失败，请查看失败清单” | 重试失败项 | TO_CONFIRM |
