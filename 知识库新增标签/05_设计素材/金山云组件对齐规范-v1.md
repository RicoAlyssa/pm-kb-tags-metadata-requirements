# 金山云星流控制台组件对齐规范 v1

> 依据 2026-07-10 实际控制台截图整理，标签原型与元数据原型共用本规范，但保持为两个独立文件。

| 区域 | 对齐规则 |
|---|---|
| 画布 | 1440×1024；页面底色 `#F5F7FA` |
| 全局顶栏 | 高 48px，白底，下边框 `#E8EBF0` |
| 产品导航 | 高 48px，白底；当前项使用品牌蓝紫色 |
| 左侧导航 | 宽 180px；选中项浅蓝底、蓝紫文字 |
| 内容边距 | 左右 24px；卡片间距 16px |
| 卡片 | 白底、4px 圆角、1px 浅灰边框，不使用重阴影 |
| 标题 | PingFang SC；页标题 20px/600，卡片标题 16px/600，正文 14px |
| 主按钮 | 高 32px，品牌色 `#526EF5`，4px 圆角 |
| 次按钮 | 高 32px，白底、灰边框、深色文字 |
| 页签 | 14px；激活态蓝紫文字与 2px 底边线 |
| 表格 | 表头浅灰底；行高 48px；操作使用文字链 |
| 输入框 | 高 32px，边框 `#D9DEE7`，4px 圆角 |
| 弹窗 | 居中，宽 560–640px；标题/正文/底部按钮三区 |
| 抽屉 | 右侧进入，宽 640–720px；用于批量编辑和复杂条件 |
| 标签胶囊 | 22px 高、3px 圆角、12px 字号；低饱和浅底色 |

## 信息架构门禁

- 不新增一级导航，不新增角色体系。
- 知识库标签入口放在知识库列表筛选、详情基础信息和文档信息页签。
- 元数据定义放在“知识库设置”；元数据赋值放在“文档信息”；检索条件放在“知识库检索”。
- 标签与元数据分别出 `.pen` 和 HTML，不在同一原型中混排。
