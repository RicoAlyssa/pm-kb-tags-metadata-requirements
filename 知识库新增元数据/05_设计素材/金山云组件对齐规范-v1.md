# 金山云星流控制台组件对齐规范 v1

本目录的元数据原型遵循 [标签目录中的统一组件规范](../../知识库新增标签/05_设计素材/金山云组件对齐规范-v1.md)，并额外遵守以下元数据专属规则：

- 新建知识库页完全保持现状；知识库创建完成后，进入“知识库设置 > 元数据定义”维护字段。
- 导入文档页完全保持现状；文档导入完成后，在文档信息页查看摘要并进行单篇或批量编辑。
- 检索页沿用现有表单组件，以“且/或 + 字段 + 运算符 + 值”的条件行表达过滤。
- P0 只展示 String 类型；其他类型用“后续版本”说明，不制作不可实现的控件。
- 不新增角色或权限页面；沿用当前 IAM 与空间权限结果。
